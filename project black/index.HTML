Sobre o projeto

Trabalho de Extensão

Envolunção da Tecnologia e as Fake News

Autor: Rian,Rodrigo,Thales,Asa Azul

Introdução

Este trabalho tem como objetivo apresentar um projeto de extensão, abordando sua importância, aplicação prática e impacto na sociedade.

Objetivo

O objetivo deste projeto é aplicar conhecimentos adquiridos em sala de aula para resolver problemas reais, contribuindo com a comunidade.

Desenvolvimento

O desenvolvimento do projeto envolve planejamento, execução e análise dos resultados. Foram utilizadas ferramentas tecnológicas para facilitar o processo.

Resultados

Os resultados demonstram a importância da integração entre teoria e prática, proporcionando benefícios tanto para os alunos quanto para a comunidade.

Conclusão

Conclui-se que projetos de extensão são fundamentais para o desenvolvimento acadêmico e social, fortalecendo o aprendizado e a cidadania.
